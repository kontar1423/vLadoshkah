# Volunteer API

## Installation

To install and run the Volunteer API, follow these steps:

1. Clone the repository:
```
git clone https://github.com/your-username/volunteer-api.git
```

2. Install the dependencies:
```
cd volunteer-api
npm install
```

3. Start the development server:
```
npm run dev
```

The API will be running at `http://localhost:4000`.

## Usage

The Volunteer API provides the following endpoints:

- `/animals`: CRUD operations for animal data
- `/shelters`: CRUD operations for shelter data
- `/users`: CRUD operations for user data

You can use a tool like Postman or cURL to interact with the API.

## API

The API endpoints are documented using Swagger UI, which can be accessed at `http://localhost:4000/docs`.
