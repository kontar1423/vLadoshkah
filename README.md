# В Ладошках

React website built with Vite and Tailwind CSS.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Fonts

Make sure to place SF Pro Rounded font files in the `public/fonts/` directory:
- SF-Pro-Rounded-Thin.woff2
- SF-Pro-Rounded-Ultralight.woff2
- SF-Pro-Rounded-Light.woff2
- SF-Pro-Rounded-Regular.woff2
- SF-Pro-Rounded-Medium.woff2
- SF-Pro-Rounded-Semibold.woff2
- SF-Pro-Rounded-Bold.woff2
- SF-Pro-Rounded-Heavy.woff2
- SF-Pro-Rounded-Black.woff2
